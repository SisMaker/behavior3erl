中文
=====
Behavior3的erlang支持库

快速开始
----
添加如下内容到**rebar.config**

    {deps, [
       ...
       {behavior3erl, {git, "https://github.com/dong50252409/behavior3erl.git"}}
    ]}.

编译
----
    $ rebar3 compile
